🎓 CampusGigAI-Powered Micro-Internship Marketplace for UniversitiesCampusGig is a specialized platform that connects university students with professor-led research projects and technical tasks. It leverages AI analysis to bridge the gap between project requirements and student skill sets, ensuring the best match for every "Gig."🌟 Key FeaturesDual-Role Authentication: Secure JWT-based login for both Professors and Students with unique dashboard views.AI Match Scoring: When a student applies, the system performs a real-time analysis of their bio and skills against the gig description, generating a percentage-based match score.Professor Control Center: Professors can post new gigs, track total applicant counts, and review individual candidate profiles.One-Click Recruitment: A streamlined interface for professors to Accept or Reject applicants, updating the student's status instantly.Interactive Activity Feed: A dynamic "Recent Activity" list that provides one-click navigation to project management pages.🛠️ Tech StackLayerTechnologyFrontendReact, TypeScript, Tailwind CSS, Framer MotionBackendNode.js, ExpressDatabase/ORMPrisma ORM, PostgreSQL/SQLiteIcons & UILucide React, Radix UI (Shadcn)
